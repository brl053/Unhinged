Instructions

To test image analysis:

  1. Ensure the Vision service is healthy (green status above)
  2. Upload an image by clicking the upload area or dragging and dropping
  3. Adjust options if needed (tags, objects, prompt, etc.)
  4. Click one of the analysis buttons to process the image
  5. Wait for the AI-generated results to appear

Analysis Types:

  • Analyze Image: Complete analysis with description, metadata, and tags
  • Describe Image: Detailed natural language description
  • Detect Objects: Identify and locate objects in the image

Troubleshooting:

  • Service Unavailable: Ensure Vision AI service is running: docker ps | grep vision
  • CORS Error: Browser blocks localhost requests. Solutions:
    • Serve this file via HTTP server: python3 -m http.server 8080
    • Test with curl: curl -X POST http://localhost:8001/analyze -F "image=@your_image.jpg"
  • Model Loading: First request may be slow as BLIP model loads
  • Check that the image file is under 10MB and in a supported format (JPEG, PNG, GIF, BMP, WebP)
  • Try refreshing the page if upload doesn't work
  • Use Chrome or Firefox for best compatibility

Alternative Testing Methods:
