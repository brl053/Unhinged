# Unhinged: Vision & Positioning (2025)

## Core Vision

Unhinged is building the operating system for the post-GUI era. We reject the assumption that graphical interfaces must be the primary way humans interact with computers. Instead, we're building a kernel-native platform where natural language is the primary abstraction layer.

## The Problem We Solve

Users today are trapped between:
- **Legacy GUI paradigms** (Windows, macOS) designed for 1990s workflows
- **Headless servers** that require terminal expertise
- **Mobile platforms** that sacrifice power for simplicity

System administration remains a specialized skill. Users cannot express intent in natural language and have the system figure out what commands to run. They must know the exact command syntax, man page sections, and piping relationships.

## Our Solution

Unhinged transforms natural language intent into orchestrated kernel commands:

```
User: "My headphone volume is too low"
System: Discovers pactl, amixer, alsamixer
System: Builds execution DAG
System: Runs commands in parallel
System: Returns results with reasoning
```

## Strategic Positioning

**Unhinged is NOT:**
- A Linux distribution (we're a platform layer)
- A replacement for existing desktops (we're complementary)
- A niche tool (we're mainstream infrastructure)

**Unhinged IS:**
- A native graphics platform with kernel-first architecture
- An intent-driven system that understands natural language
- A bridge between human intent and system capability
- The foundation for next-generation computing interfaces
