BACKGROUND & CONTEXT

The Unhinged project represents a fundamental shift in operating system design philosophy. We are building a kernel-native platform that treats natural language as the primary user interface, with command orchestration as the execution engine.

This memorandum documents the current state of the project, technical decisions made to date, and the strategic direction for future development. All information contained herein reflects the consensus of the technical leadership team and represents the official position of the organization.

KEY INITIATIVES

1. Command Orchestration Pipeline - Natural language to kernel command translation
2. Semantic Search Engine - Vector-based discovery of relevant system commands
3. DAG Construction & Execution - Parallel execution of interdependent commands
4. Jinja2 Prompt Templating - Reusable organizational communication framework
5. Safety & Audit Framework - Confirmation, rollback, and complete audit trails
