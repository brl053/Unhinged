Instructions

To test voice recording:

  1. Ensure the Whisper service is healthy (green status above)
  2. Grant microphone permissions when prompted by your browser
  3. Click the microphone button to start recording
  4. Speak clearly into your microphone
  5. Click the stop button to end recording
  6. Wait for the transcription result to appear

Troubleshooting:

  • If service is unhealthy, ensure Docker containers are running: docker compose -f docker-compose.simple.yml ps
  • If microphone access is denied, check browser permissions
  • Try refreshing the page if recording doesn't work
  • Use Chrome or Firefox for best compatibility
